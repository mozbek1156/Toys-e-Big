## ✨ Key Features

- 📱 **Fully Responsive**: Optimized for all devices—from smartphones to high-resolution desktops.
- 🎨 **Modern UI/UX**: Dark-mode aesthetic with vibrant gradients and glassmorphism.
- ⚡ **Performance Optimized**: Vanilla JavaScript implementation for lightning-fast load times.
- 🔍 **SEO Ready**: Semantic HTML5 structure for better search engine indexing.
- 🎬 **Smooth Animations**: Intersection Observer-powered entrance animations and scroll-based parallax effects.
- 📈 **Interactive Counters**: Dynamic statistics showing project completion and satisfaction rates.

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mozbek1156/Toys-e-Big.git
   ```
2. **Open the project**:
   Navigate to the directory and open `index.html` in your favorite browser.
3. **Customize**:
   Edit `style.css` to update themes/colors and `script.js` for custom logic.
